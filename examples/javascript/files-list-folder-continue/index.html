examples / filesListFolderContinue

This example fetches the contents of your root Dropbox directory with a limit of 3. And if the result's ListFolderResult.has_more field is true. call list_folder/continue with the returned ListFolderResult.cursor to retrieve more entries. It uses the Dropbox.filesListFolderContinue() method [docs]

    To obtain an access token for quick testing, you can go to API Explorer click the "Get Token" button on the top right, copy the token it creates and then paste it here.
